Project Overview

This project is a comprehensive banking application with CRM functionalities built using Angular and Firebase. It includes features for user authentication, role management, account management, transfers, and more. The application also utilizes Material Design for a modern and responsive user interface.

Key Features

  • User Authentication
    • Firebase Authentication: Users can log in using Firebase authentication.
    • Guest Login: Guests can log in without creating an account. Guest data is deleted upon logout.
  • Role Management
    • Role Change: Guests can change their role to user, admin, or manager.
    • Role-Based Access: Different functionalities are available based on the user's role.
  • Account Management
    • Create New Accounts: Users can create new accounts.
    • View and Edit Accounts: Users can view and edit account details.
    • Delete Accounts: Users can delete accounts.
  • Transfers
    • Send Money: Users can send money to other accounts.
    • Move Money: Users can move money between their own accounts.
    • Transfer Fees: Transfer fees are applied based on the type of transfer.
    • Transfer Details: Users can view details of their transfers.
  • Automatic Interest Payment
    • Interest Calculation: Interest is automatically calculated and credited to user accounts.
  • Dashboard
    • User Dashboard: Displays user information, account balances, and recent transfers.
    • Admin Dashboard: Displays bank statistics, user growth, and other admin-specific information.
  • Profile Management
    • Profile Picture: Users can upload and update their profile picture.
    • Edit Profile: Users can edit their profile information, including email and password (with email verification for non-guests).
  • Statistics and Charts
    • Income and Expenses Chart: Displays a chart of income and expenses.
    • User Growth Chart: Displays a chart of user growth over time.
    • Transfers Chart: Displays a chart of transfers.
  • Notifications
    • Snackbar Notifications: Users receive notifications for successful actions and errors.
    • Dialog Service: Users receive dialog notifications for important messages and errors.
  • Dialogs
    • Edit Account Dialog: Allows users to edit account details.
    • Send Money Dialog: Allows users to send money to other accounts.
    • Move Money Dialog: Allows users to move money between their own accounts.
    • Edit Interest Rate Dialog: Allows admins to edit the bank's interest rate.
    • Edit Transaction Fee Dialog: Allows admins to edit the bank's transaction fee.

Additional Functionalities

  • Email Verification: When a non-guest user changes their email, the new email must be verified.
  • Loading Spinner: A loading spinner is displayed during data fetching and processing.
  • Error Handling: Comprehensive error handling with user-friendly messages.
